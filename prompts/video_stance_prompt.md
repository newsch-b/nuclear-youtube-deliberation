# 影片立場標記提示詞（Video Stance Annotation Prompt）

**模型**：claude-sonnet-4-6  
**任務**：先摘要、後判定（Two-stage: summarize → classify）  
**說明**：口語逐字稿含大量口頭禪與離題內容，先摘要書面論點，再判定核能立場，可降低形式雜訊干擾。

---

## Prompt

```
以下是一段 YouTube 影片逐字稿，主題與台灣核能議題相關。

【任務一：重點摘錄】
從逐字稿中找出最具代表性的論點，條列整理：
- 每點一句話，使用書面語，保留原始立場與語氣，不要中立化
- 最多 3 點；若只有一個核心論點就只寫 1 點
- 若段落無實質論點（閒聊、過場、與核能無關），summary 填 "no_claim"

【任務二：立場判定】
根據整理出的論點判斷整段的核能立場：
- "pro"：支持核能
- "anti"：反對核能
- "neutral"：純陳述事實、或立場不明確

【輸出格式（JSON）】
{"segment_id": "...", "summary": ["論點一", "論點二"], "stance": "pro/anti/neutral", "stance_confidence": "high/mid/low"}
```

---

## 範例

**輸入**（2025年8月7日公視新聞網公投辯論片段）：

> 所以我們今天在看核電，不是只有單純的用電需求，它還牽涉到能源安全，牽涉到國家安全。現在請反方的代表人曾文生先生發言，時間12分鐘，請發言。好，那個正方代表剛才提到非常多就是能源安全國家安全的問題，但是他質疑說我們的燃煤機組為什麼不拆。那我也可以跟各位說明，燃煤的儲量剛才提到了，它是超過30天，事實上它在整個煤廠的調度狀況好的狀況下，還可以有更長的時間。所以把燃煤機組留下來做國安戰備的機組，這個我們在環評會上一而再再而三做說明。

**輸出**：

```json
{
  "segment_id": "pts_debate_20250807_seg03",
  "summary": ["燃煤機組儲量超過30天，可作為國安戰備機組，無需拆除"],
  "stance": "anti",
  "stance_confidence": "high"
}
```

---

## 設計說明

- **信心程度**（`stance_confidence`）作為輔助指標，低信心標注優先進行人工核對
- **分段策略**：長文本以 500 字元為上限分段，在 450–500 字元區間尋找句尾標點截斷
- **信度**：人機一致性 Accuracy = 90.3%，Cohen's κ = 0.85；Claude vs GPT-4.1-mini κ = 0.84
